# 📊 Data Cleaning & Reporting Automation

## 📌 Project Overview

This project automates the process of cleaning raw datasets and generating reports. It handles missing values, duplicate records, incorrect data types, and inconsistent text formatting while producing summary reports and visualizations. A Streamlit dashboard provides an interactive interface for uploading and cleaning datasets.

---

## 🚀 Features

- Automated data cleaning
- Missing value handling
- Duplicate removal
- Date conversion
- Text standardization
- Statistical summary generation
- Data visualization
- Streamlit dashboard
- Download cleaned dataset

---

## 🛠 Technologies Used

- Python
- Pandas

- Streamlit

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Data-Cleaning-Reporting-Automation.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the automation pipeline:

```bash
cd src
python automation.py
```

Run the Streamlit dashboard:

```bash
streamlit run app.py
```
